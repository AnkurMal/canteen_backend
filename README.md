# Steps to run the project

1. [Install UV](https://docs.astral.sh/uv/getting-started/installation/)
2. run `uv python install`
3. run `https://github.com/AnkurMal/canteen_backend.git`
4. run `code canteen_backend/`
5. open terminal (in vscode) and run `fastapi dev`
6. go to [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs) for api docs
